
CI Windows Build (GitHub Actions)
=================================
1) Create a new GitHub repository and commit all files from this folder.
2) Push to 'main' (or 'master').
3) Go to the 'Actions' tab -> 'Windows EXE Build' workflow -> wait for it to finish.
4) Download the built EXE from the workflow 'Artifacts': sensor_mode_tool2-windows-exe.
Local build (optional): run PowerShell:  ./Build.ps1
